# Python Path Finder on Grid

## About

A Python Path Finder Simulator that finds a path from a **Start (S)** point to an **End (E)** point on a grid while avoiding obstacles (**X**).
This project simulates basic robot path planning using a grid-based environment.

---

## ✨ Features

✔️ Grid-based environment
✔️ Start and End position detection
✔️ Obstacle avoidance
✔️ Path finding using DFS algorithm
✔️ Visual path display using `*`

---

## 🛠️ Tech Used

* Python
* Depth First Search (DFS)
* Recursion
* Sets
* 2D Lists (Grid)

---

## 🚀 How to Run

1. Download the project
